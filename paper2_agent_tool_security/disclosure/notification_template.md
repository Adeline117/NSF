## Responsible Disclosure Notice

Dear maintainers,

We are security researchers from the University of Illinois conducting an NSF-funded study on AI agent tool interface security (NSF Award #2425501). We identified potential vulnerabilities in your repository as part of a systematic analysis of 138 tool server implementations across 4 protocol families (MCP, OpenAI Function Calling, LangChain, and Web3-native modules).

This report follows responsible disclosure practices. Key details:

- **Disclosure window:** 90 days from the date of this report
- **Public disclosure deadline:** 2026-07-07
- **Methodology:** Automated static analysis + dynamic tool-interface testing (TCPI framework)
- **Paper:** Under submission to IEEE S&P 2027

We welcome your response and are happy to assist with remediation. If you need additional time to address the findings, please let us know and we will extend the disclosure timeline accordingly.

**What we ask:**
1. Acknowledge receipt of this report
2. Let us know if you plan to address the findings
3. Contact us if any finding is a false positive so we can correct our dataset
